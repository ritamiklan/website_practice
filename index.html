About this site

This website is for practicing my HTML and CSS skills. This is a final project for the Orientation to Digital Services course. This website features some resources for studying, my next steps towards becoming a software engineer, and a showcase of different page layouts and techniques in CSS.

Studies

I decided to pursue a career in software engineering. I applied to Haaga-Helia and started a Degree Program in Business and Information Technology in 2020. This is a career change for me, and a new field of interest. I am excited to see what the future holds!

Project description

The goal is to create a fully responsive website with HTML and CSS, and without using any frameworks, preprocessors, or external libraries. This website represents my current knowledge of HTML and CSS. The main page uses Flexible Box layout. The different pages will be using different layouts.

Points of evaluation:

  1. HTML and CSS validity
  2. Different layouts
  3. Semantics
  4. Usability
  5. Content
  6. Colors, fonts, and so on
